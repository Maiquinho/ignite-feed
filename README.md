# Ignite Feed 

Project developed on [Rocketseat](https://www.rocketseat.com.br/) Ignite Bootcamp (Trail 2022) focused on study the main concepts of Reactjs, like:


- JSX
- TSX
- Components
- Properties
- CSS Modules
- React Engines
    - States
    - Immutability
- TypeScript
